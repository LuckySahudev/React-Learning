1. To use Tailwind css
    
    -> go the web page tailwind css
    -> click on Get Started
    -> click on Using vite
    -> follow steps 
    -> start using tailwind 

    Note :
        1. Import in vite.config.ts

            import tailwindcss from '@tailwindcss/vite'

            plugins: [
                tailwindcss(),
            ],

        2. Import in index.css

            @import "tailwindcss";
